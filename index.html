Description

Esc | adalah escape untuk keluar dari menu atau membatalkan suatu proses

F1 | Biasanya dibuat untuk membuka menu bantuan (Help).

F2 | Untuk mengganti nama (Rename) file atau folder.

F3 | Untuk mencari (Search) file atau teks.

F4 | Membuka drop-down address bar / Alt + F4 untuk menutup aplikasi.

F5 | Untuk menyegarkan halaman (Refresh), misalnya di browser.

F6 | Memindahkan fokus ke elemen di aplikasi, seperti address bar.

F7 | Mengecek ejaan dan tata bahasa (di Microsoft Word).

F8 | Masuk ke Safe Mode saat booting (Windows versi lama).

F9 | Mengirim dan menerima email di Outlook, atau menghitung ulang di Excel.

F10 | Mengakses menu bar. Shift + F10 = klik kanan.

F11 | Mengaktifkan atau keluar dari mode layar penuh (Fullscreen).

F12 | Membuka "Save As" di Microsoft Word atau Developer Tools di browser.

PrtSc | Mengambil screenshot seluruh layar dan menyimpannya ke clipboard.

Delete | Menghapus teks di depan kursor, atau file yang dipilih (ke Recycle Bin).

Delete | Menghapus teks di depan kursor, atau file yang dipilih (ke Recycle Bin).

Insert | Mengaktifkan mode overwrite: saat mengetik, huruf di depan kursor akan tertimpa.

Backspace | Menghapus karakter di sebelah kiri kursor (mundur satu huruf)

Shift | Membuat huruf besar (kapital), dan mengakses simbol atas di tombol (misal !, @, #).

Home | Memindahkan kursor ke awal baris.

Page Up | Menggeser tampilan layar ke atas satu halaman.

Page Down | Menggeser tampilan layar ke bawah satu halaman.

End | Memindahkan kursor ke akhir baris.

Fn | Tombol fungsi tambahan (khusus laptop), untuk akses volume, brightness, dsb.

Windows | Membuka Start Menu, atau dikombinasikan dengan tombol lain untuk shortcut (misal Win + E).

Ctrl | Digunakan bersama tombol lain untuk perintah cepat (Copy = Ctrl + C, Paste = Ctrl + V).

Alt | Kombinasi dengan tombol lain untuk fungsi khusus (Alt + Tab = pindah antar jendela).
